
# Todo App

To Do App (from 03-21-2023 to ...)
## Tech Stack

**Client:** Svelte, Vite, Routify, TailwindCSS

**Server:** Node


## Features

- Auth
- Routing
- CRUD with Svelte and Supabase
## Installation
- Clone this repository
```bash
git clone https://github.com/Firmansyahp16/app
```
- Open folder
```bash
cd app
```
- Install with pnpm

```bash
  pnpm install
```
- Run development
```bash
pnpm run dev
```
